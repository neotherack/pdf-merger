# pdf-merger


## Installation:

### Windows
- python -m venv venv
- pip install -r requirements.txt

### Linux
- python3 -m venv venv
- pip3 install -r requirements.txt


## Execute
### Windows
- python main

### Linux
- python3 main


## Usage:
1) Set your working folder
2) Tell which files will be merged (introduce "_end_" to stop asking source files)
3) Set a name for the resulting file
4) Wait
